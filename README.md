# Solutions & tests for WD201 assignmets

## Format for `report.json`

```json
{
  "version": 0,
  "grade": "skip/accept/reject",
  "status": "success/failure",
  "feedback": [
    {
      "key": "key_1_for_feedback_text",
      "variables": {
        "text_variable_key_1": "text_variable_value_1",
        "text_variable_key_2": "text_variable_value_2"
      }
    },
    {
      "key": "key_2_for_feedback_text"
    },
  ],
  "report": [
    {
      "key": "key_1_for_report_text",
      "variables": {
        "text_variable_key_1": "text_variable_value_1",
        "text_variable_key_2": "text_variable_value_2"
      }
    }
  ]
}
```
